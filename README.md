

## Test na telefonu přes lokální Wi-Fi

1. Připoj Mac i telefon ke stejné Wi-Fi.
2. Otevři projekt ve VS Code a spusť **Live Server**.
3. Ve VS Code zkontroluj adresu serveru; projekt je nastavený na port `5500` a přístup přes lokální IP.
4. Na telefonu otevři v Safari adresu ve tvaru `http://IP_ADRESA_MACU:5500`.
5. Ověř tažení vstupních kamenů, stabilitu stránky a čitelnost desky.

Lokální IP Macu lze zjistit v Nastavení systému → Wi-Fi → Podrobnosti → TCP/IP.
